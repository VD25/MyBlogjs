Project on console

Visual Studio Code is a freeware source-code editor made by Microsoft for Windows. We can download Visual Studio Code by going to its website and we can start coding by making a folder containing the files in which the write the coding language.

Console screen logs the same sorts of information as the Web Console - network requests, JavaScript, CSS, and security errors and warnings, and messages explicitly logged by JavaScript code. To open the console screen we can right click anywhere on the tab screen and click on inspect. Then we can click on the console tab to see the console screen. To write something on the console screen we can write the code- console.log("text"). For example- console.log("hello")
